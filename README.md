# Excel_Data-Analysis_Dashboard
I Developed MD Store Annual Report 2025 is an interactive sales dashboard that analyzes yearly e-commerce performance across multiple channels. It provides insights into sales trends, top-performing regions, customer demographics, order status, and channel-wise performance to support data-driven business decisions.
## Project Objective
To analyze annual e-commerce sales data using an interactive dashboard in order to identify sales trends, customer behavior, and channel performance, supporting informed and data-driven business decisions.
